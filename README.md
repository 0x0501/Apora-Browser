# Apora-Browser

Browser extension for using [Apora](https://apora.sumku.cc) on browsers.

## Development

Clone this repository, create a `web-ext.config.ts` and config Chrome, Firefox,
and Edge binary path (check it up at [WXT Browser-Startup](https://wxt.dev/guide/essentials/config/browser-startup.html)), after that running the command below:

```bash
bun run dev
```
